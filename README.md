# LEDeej
Deej but with RGB !

## Hardware
- Arduino Uno
- WS2812
- 75 mm 10 K Mixeur Fader
- JR Connector
- 4x M3x30 Screw
## PIN used :
- LED : 2, 3, 4, 5, 6
- Slider : A0, A1, A2, A3, A4
